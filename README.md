# 🛡️ A.T.I.C.E - Automated Threat Intelligence Correlation Engine

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![Flask](https://img.shields.io/badge/Flask-2.3-green)
![License](https://img.shields.io/badge/License-MIT-orange)

A professional **Security Operations Center (SOC) Dashboard** that simulates real-time threat detection, MITRE ATT&CK mapping, and threat intelligence correlation.

## ✨ Features

- **Real-time Attack Monitoring** – Live alerts every 2 seconds
- **MITRE ATT&CK Integration** – Maps attacks to techniques (T1110, T1190, etc.)
- **Threat Intelligence Simulation** – IP reputation scoring
- **Client-side Blocking** – Block IPs with browser localStorage
- **Investigation Reports** – Simulated threat intelligence for each IP
- **6 Dashboard Pages** – Dashboard, Alerts, Threat Feed, MITRE, Reputation, Blocked IPs
- **Responsive Design** – Dark theme, professional UI
